Проект - площадка для инвесторов и заемщиков.
В проекте могут работать физ. лица и юр. лица.
То есть пользователи проекта делятся на заемщиков и инвесторов. А также, делятся по форме собственности.
Проекты написаны на ASP.
Проект лежит на DropBox.
Проект запускается в Visual Studio.

Все запросы отправляются на сервак, в ответ приходит JSON модель. По ней рендерятся данные.
Строгие правила верификации всех форм.

На KENDO идут верификации по заполнению дат. Плюс подключены виджеты на числовые поля, для отображения сумм.
